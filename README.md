#  Road Accident Dashboard

An interactive dashboard for analyzing road accident data using **HTML**, **CSS**, and **JavaScript**.

##  Overview
This project visualizes road accident data to identify trends, compare statistics across states, and analyze year-wise variations. It aims to make data interpretation simple and insightful through interactive charts and clean design.

##  Features
- Year-wise accident data comparison  
- State-wise accident visualization  
- Interactive charts using JavaScript  
- Responsive and easy-to-use interface  
- Hosted publicly using **GitHub Pages**
